Statements (Level-Control-Structures)

Created2023-04-26|Updated2023-04-26|CSCI3180
|Post View:

Two-Way Selection Statements

  • provides the means of choosing between two or more paths of execution

  • General Form:

    • the control expression must be Boolean in most of the languages
    • some other languages like C accepts numeric

  • Clause Form:

    • the then-clause and else-clause can be signle statement or compound statements in most of the languages
    • Perl only accepts compound statements
    • Python and Ruby: clauses are statement sequences

Nesting Selectors

  • Dangling else problem

    • In C/C++/Java, else matches with the nearest previous if

  • In Perl, this is actually enforced

  • In some languages, clauses are statement sequences which are ended by a reserved word end

  • In Python,

Multiple-Way Selection

  • allow the selection of one of any number of statements or statement groups

  • Switch statement in C, C++, and Java

    • Control expression and case constants can only be integers

    • default clause is for unrepresented values

    • Placement

      • switch-case has no restructions on the placement of the case expression
  • unconditional branch in C#
    • C# allows unconditional branch like goto and break

  • Multiple Selector in two-way selectors:

    • the connection of two-way selections using else if

    • but the structure is more look like this (which is not an actual multiple-way selection)

    • In Python, elif is a single special keyword and enables actual multiple-way selection

  • Implementation
    • General approach
      • multiple conditional branches
      • linear search of a simple table of case values
    • Optimized approaches
      • array
      • hash table

Counter-Controlled Loops

  • has a loop variable, and a means of specifying the initial, condition, and step

  • C-based language Syntax: for ([initial] ; [condition] ; [step]) body

    • the [initial], [condition] and [step] can be whole statements or even statement sequences(statement seperated by ,)

    • It can even has no statements

    • It is legal for C++ to go back to the loop body using goto

Logic-Driven Loops

  • the repetition control is based on a Boolean expression

  • Pretest Loop

    • check the loop condition before the first iteration
    • while loop

  • Posttest Loop

    • check the loop condition after the first iteration
    • do-while loop

  • It is also legal for C++ to go back to the loop body using goto

Data-Structure-Based Loops

  • the number of elements in a data structure controls loop iteration

  • call to an iterator function or a callable object that returns the next element in some chosen order, if there is one; else loop is terminated

  • Implement using pointer in C

  • Standard Template Library in C++

    • instead of dealing with pointer, STL iterators can simply using the deference operator * and increment operators ++

  • Java Implementation

    • the class that implements the Iterable interface (ArrayList)

    • forEach method

  • Python Implementation

    • The loop variable takes on the values specified in the given sequence

    • else clause is execute if the loop terminates normally

    • similate counter-controlled loop

User-Located Loop Control Mechanisms

  1. unconditional unlabeled exits (break in most of the languages, last in Perl)

    • exit the loop

  2. unlabeled control statement (continue)

    • skips the the remainder of the current iteration, but does not exit the loop

  3. Labeled exits

    • consider the break can only exist for the current loop but cannot exit the outer loop
    • we can label the outer loop and then use break label; to exit the outer loop
    • If goto is legal, then simply use goto to exit the outer loop
