CSCI3230 - Fundamental Concepts in AI

Created2022-09-10|Updated2022-09-10|ESTR3108
|Post View:

Machine Learning Path

Machine Learn

  1. Acquire the skill with experience computed from data
  2. Feed data (e.g. text documents, images, sound recordings, DNA sequences) to AI systems
  3. Model Prediction: The skill of a machine can be illustrated by its prediction

Examples

1. AI in healthcare

  • Data: CT images with lung nodules
  • Skill: Automatically detect these abnormalities in CT images
  • Prediction: The coordinates and width and height of a bounding box

2. AI in autonomous driving

  • Data: Images with pedestrians and cars
  • Skill: Automatically positioning pedestrians and cars on the road
  • Prediction: Pixel-wise masks of pedestrians and cars on the images

3. AI in languages

  • Data: Some language pairs
  • Skill: Automatically translate the language
  • Prediction: The translated texts

4. AI in robot

  • Data: The environment of robot
  • Skill: Automatic pancake flipping
  • Prediction: The next action of robot

Learn from Data

Learning Process

  • We use some learning algorithm to train the machine how to predict.
  • The machine will then construct a model based on the certain learning algorithm.
  • Learn the model with observed samples in data sets.

Data Split

  • Split the data randomly into training, validation and test data sets:
    Traning data set: helps machines to learn and make predictions.
    Validation data set: tunes the hyper-parameters of the model.
    Test data set: provide an unbiased evaluation of a final model fit on the training dataset.
  • Train the machine based on the training data set and then the machine can process the test data set.
  • A larger test set gives more accurate assessment of performance.

Cross Validation

  • A method used to evaluate AI models on a small-scale data set.
  • Split the dataset into k groups.
  • For each unique group:
    • take a group as a holdout or test data set.
    • take the remaining groups as the training data set.
  • Summarize the skill of the model using all the model evaluation scores.

Evaluate Model Performance

Machine Learning Classifier

  • A tool for evaluating the model performance

  • Classifies the data into positive samples (Class +) and negative samples (Class -).

  • Accuracy (the fraction of correct predictions) = $ (TP + TN) \over (TP + FP + TN + FN) $

  • Precision (the accuracy of the positive predictions) = $ (TP) \over (TP + FP) $

  • Recall / Sensitivity (the accuracy for positive class) = $ (TP) \over (TP + FN) $

Confusion Matrix

  • a specific table layout that allows visualization of the performance of an algorithm.

Type of Learning

1. Supervised Learning

  • A machine learning task, the machine learns a function that maps an input to an output based on the examples of input-output pairs.

  • Provide the labeled data. Then, the machine can learn from the characters of the objects and labels.

  • Application: objective classification, text translation.

2. Unsupervised Learning

  • A type of algorithm, that the machine learns from unlabeled data without knowing the categories.

  • Just provide the unlabeled data (raw data). Then, the machine can learn from the characters of the data on its own.

  • Application: determining customer segments in marketing data.

3. Reinforcement Learning

  • An area of machine learning, involves how should the intelligent agents take actions in an environment, in order to maximize the cumulative reward.

  • Application: AI in playing games, real-time bidding.
