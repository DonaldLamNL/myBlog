Hadoop Cluster Setup

Created2023-09-18|Updated2023-09-18|ESTR4300
|Post View:

Hadoop Configuration

  • Hadoop use a collection of configuration properties and values
  • Configurations can be either defined in XML files (offline), or defined in programs (online)
    • Resources are XML files that define properties/values
  • XML format with name, value, description (optional): Notice: Properties that are marked final cannot be overridden in later definitions
  • Add the resources in the program:

Configuration Files

Filename Description
core-site Configuration settings for Hadoop Core, such as I/O settings that are common to HDFS and MapReduce
hdfs-site Configuration settings for Hadoop Core, such as I/O settings that are common to HDFS, MapReduce, and YARN
mapred-site Configuration settings for MapReduce daemons: the job history server
yarn-site Configuration settings for YARN daemons: the resource manager, the web app proxy server, and the node managers

Setup

Create VMs

  • Set up 4 virtual machine instances in Google Cloud Platform using Ubuntu 20.04 LTS Operating system and 100GB balanced persistent disks with names namenode-1, datanode-1, datanode-2, datanode-3

Create Users

  • For all instances, create a user named hduser:
    sudo -i
    adduser hduser (pwd=hduser)

  • Advise the sshd configuration:
    vi /etc/ssh/sshd_config

    service sshd restart

  • Connect the instances:
    vi /etc/hosts

Generate ssh key

  • Setup a ssh connection in Master instance:
    su hduser
    cd ~
    ssh-keygen -t rsa -P ""

  • Copy the ssh id to the Worker instances:
    ssh-copy-id -i /home/hduser/.ssh/id_rsa.pub hduser@datanode-1
    ssh-copy-id -i /home/hduser/.ssh/id_rsa.pub hduser@datanode-2
    ssh-copy-id -i /home/hduser/.ssh/id_rsa.pub hduser@datanode-3
    ssh-copy-id -i /home/hduser/.ssh/id_rsa.pub hduser@namenode-1

  • Authorize:
    chmod 0600 ~/.ssh/authorized_keys

Hadoop and Java Installation

  1. Install Hadoop:
    cd /opt/
    wget https://archive.apache.org/dist/hadoop/core/hadoop-2.9.2/hadoop-2.9.2.tar.gz
    tar zxf hadoop-2.9.2
    mv hadoop-2.9.2 /usr/local/hadoop

  2. Install Java:
    wget --no-cookies --no-check-certificate --header "Cookie: gpw_e24=http%3A%2F%2Fwww.oracle.com%2F; oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/jdk-8u131-linux-x64.tar.gz
    tar xzf jdk-8u131-linux-x64.tar.gz
    cd /opt/jdk1.8.0_131
    update-alternatives --install /usr/bin/java java /opt/jdk1.8.0_131/bin/java 2
    update-alternatives --config java
    java -version

  • Openjdk8 (/usr/lib/jvm/java-8-openjdk-amd64)

Configure Hadoop

Do the cofiguration in the Master instance

  1. Make directory:
    chown -R hduser:hduser /usr/local/hadoop
    mkdir -p /usr/local/hadoop_store/tmp
    mkdir -p /usr/local/hadoop_store/hdfs/namenode
    mkdir -p /usr/local/hadoop_store/hdfs/datanode
    mkdir -p /usr/local/hadoop_store/hdfs/secondarynamenode
    chown -R hduser:hduser /usr/local/hadoop_store

su hduser

  1. Configuate hdfs-site.xml:
    vi /usr/local/hadoop/etc/hadoop/hdfs-site.xml

  2. Configuate core-site.xml:
    vi /usr/local/hadoop/etc/hadoop/core-site.xml

  3. Configuate mapred-site.xml:
    vi /usr/local/hadoop/etc/hadoop/mapred-site.xml

  4. Configuate yarn-site.xml:
    vi /usr/local/hadoop/etc/hadoop/yarn-site.xml

  5. Add java path in hadoop-env.sh file:
    echo 'export JAVA_HOME=/opt/jdk1.8.0_131' >> /usr/local/hadoop/etc/hadoop/hadoop-env.sh
    echo 'export JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64' >> /usr/local/hadoop/etc/hadoop/hadoop-env.sh (openjdk-8-jdk)

  6. add hadoop and java path in the .bashrc file:
    vi /home/hduser/.bashrc

    source ~/.bashrc

Copy to Workers and Setup

  1. Copy Java:
    su hduser
    cd /opt/
    scp jdk-8u131-linux-x64.tar.gz datanode-1:/home/hduser/
    scp jdk-8u131-linux-x64.tar.gz datanode-2:/home/hduser/
    scp jdk-8u131-linux-x64.tar.gz datanode-3:/home/hduser/

  2. Copy hadoop:
    cd /usr/local
    scp -r hadoop datanode-1:/home/hduser/
    scp -r hadoop_store datanode-1:/home/hduser/
    scp -r /home/hduser/.bashrc datanode-1:/home/hduser/

    scp -r hadoop datanode-2:/home/hduser/
    scp -r hadoop_store datanode-2:/home/hduser/
    scp -r /home/hduser/.bashrc datanode-2:/home/hduser/

    scp -r hadoop datanode-3:/home/hduser/
    scp -r hadoop_store datanode-3:/home/hduser/
    scp -r /home/hduser/.bashrc datanode-3:/home/hduser/

Setup Hadoop and Java in Workers

  1. Move folders:
    sudo -i

  2. Setup Java:
    cd /opt
    tar xzf jdk-8u131-linux-x64.tar.gz
    cd jdk1.8.0_131
    update-alternatives --install /usr/bin/java java /opt/jdk1.8.0_131/bin/java 2
    update-alternatives --config java

Format namenode and start all Hadoop services (namenode)

  1. Setup datanode:
    vi /usr/local/hadoop/etc/hadoop/slaves

  2. Format namenode:
    su hduser
    hadoop namenode -format

  3. Start storage and yarn services:
    start-dfs.sh
    start-yarn.sh
