MongoDB 交互

數據庫

  1. 關係型數據庫 (SQL)

    • sql語句增刪改查操作
    • 保持事務的一致性

  2. 非關係型數據庫 (No-SQL)

    • 不用sql語句
    • 輕量、高效、自由

  3. 數據庫概念：

    SQL術語 MongoDB術語 說明
    database database 數據庫
    table collection 數據庫表
    row document 數據紀錄文檔
    column field 數據字段
    index index 索引
    table.joins 表連結，MongoDB不支持
    primary key primary key 主鍵，id
